## The Daily Brief

A real-time threat intelligence dashboard that aggregates, enriches, and visualises cybersecurity intelligence from 20+ authoritative sources.

**Live at [intel.mylr.sh](https://intel.mylr.sh)**

### Features

- **Feed aggregation** — RSS/Atom and JSON API feeds across vulnerability disclosures, malware campaigns, APT activity, data breaches, and government advisories
- **IOC extraction** — automatic identification of IPs, domains, file hashes, CVE IDs, and MITRE ATT&CK technique IDs from article content
- **DEFCON scoring** — automated threat level calculation weighted across critical articles, active exploits, APT activity, and malware prevalence
- **Interactive threat map** — geolocated incidents on a Leaflet.js map with clustering and heatmap layers
- **AI Daily Brief** — executive summary generated by an AI agent (Claude API with local Ollama fallback) from the top prioritised intelligence items
- **Archive** — monthly rolling archive with deduplication and historical browsing
- **Security-first frontend** — all dynamic rendering uses safe DOM construction (`textContent` only, no `innerHTML` with external data)

### Feed Sources

| Category | Sources |
|----------|---------|
| Vulnerability | CISA KEV, Microsoft MSRC, Google Project Zero, SecurityWeek |
| Malware | The Hacker News, abuse.ch (URLhaus, ThreatFox, MalwareBazaar) |
| APT | Palo Alto Unit 42, Cisco Talos |
| Breach | BleepingComputer, The Record by Recorded Future |
| Advisory | UK NCSC, CERT-EU, SANS Internet Storm Center |
| Research | Krebs on Security, CyberScoop, SentinelOne |

### Tech Stack

- **Backend** — Python 3, feedparser, requests
- **AI** — Anthropic Claude API (primary), Ollama (local fallback)
- **Frontend** — Vanilla JS, Leaflet.js, CSS Grid
- **Enrichment** — AlienVault OTX, VirusTotal (optional)
- **Server** — nginx, cron (30-minute update cycle)

## Deployment

### Quick deploy (EC2 / Linux)

```bash
git clone https://github.com/x0mylr/mylr.sh
cd mylr.sh/intel
bash quick-deploy.sh
```

### Manual install

```bash
bash intel/install-daily-brief.sh
```

The installer sets up a Python virtual environment, nginx, and a cron job that runs the aggregator every 30 minutes.

### Running the aggregator manually

```bash
cd intel
python3 aggregate_feeds_enhanced.py
# Output written to intel/output/feed_data.json
```

## AI Brief

The AI agent generates an executive threat summary after each aggregation run. It is **disabled by default**.

To enable, set `"enabled": true` in the `"ai"` section of `platform_config.json` and provide an Anthropic API key via the `ANTHROPIC_API_KEY` environment variable. If Claude is unavailable, the agent falls back to a locally running [Ollama](https://ollama.com) instance.

---

## Optional Enrichment

API keys for threat intelligence enrichment can be configured in `intel/api_keys.json` (see `api_keys.json.example` for the expected structure). All enrichment integrations are optional — the platform runs fully without them.
